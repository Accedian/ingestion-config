# Dictionary Uploader

Uploads ingestion dictionaries (generated by `generate_telegraf_configs.go`) to PCA cloud.

## Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `BASE_URL` | `https://pca.kajar.npav.accedian.net` | Base URL of the PCA API |
| `AUTHORIZATION_HEADER` | `Bearer XXXXXXX` | Authorization header (Bearer token) - see [OAuth 2.0 Client API Access Guide](https://docs.accedian.io/docs/configuring-oauth-20-clients-for-api-access?highlight=oauth) |
| `DICTIONARIES_PATH` | `./generated_dictionaries` | Path to dictionaries directory |
| `INSECURE_SKIP_VERIFY` | `false` | Skip TLS certificate verification (for self-signed certs) |

## Usage

```bash
# Generate dictionaries first
cd /path/to/task
go run generate_telegraf_configs.go \
  -csv Kpi_calc_Kpicatalog-updatedGrouping.csv \
  -output ./generated \
  -dictionaries ./generated_dictionaries

# Upload to cloud
export BASE_URL="https://pca.kajar.npav.accedian.net"
export AUTHORIZATION_HEADER="Bearer your-token-here"
export DICTIONARIES_PATH="./generated_dictionaries"
export INSECURE_SKIP_VERIFY="true"  # if using self-signed certs

cd dictionaryuploader
go run main.go
```

## What It Does

1. **Loads** all JSON dictionary files from `DICTIONARIES_PATH`
2. **Fetches** each corresponding dictionary from the server
3. **Compares** local vs remote (ignoring CouchDB metadata like `_rev`)
4. **Creates or Updates** dictionaries that differ

## API Endpoints Used

- `GET /api/v3/ingestion-dictionaries/{id}` - Fetch existing dictionary
- `POST /api/v3/ingestion-dictionaries` - Create new dictionary
- `PUT /api/v3/ingestion-dictionaries/{id}` - Update existing dictionary

## Prerequisites

- Go runtime (minimum version: **1.16**, tested with go1.25.7)
