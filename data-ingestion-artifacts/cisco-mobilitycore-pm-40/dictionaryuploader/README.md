# Dictionary Uploader

Uploads ingestion dictionaries (generated by `generate_telegraf_configs.go`) to PCA cloud.

## Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `BASE_URL` | `https://pca.kajar.npav.accedian.net` | Base URL of the PCA API |
| `AUTHORIZATION_HEADER` | `Bearer XXXXXXX` | Authorization header (Bearer token) - see [Authenticate to obtain an Authorized Token](https://api.accedian.io/session.html#section/Quick-setup/Authenticate-to-obtain-an-Authorized-Token) |
| `DICTIONARIES_PATH` | `./generated_dictionaries` | Path to dictionaries directory |
| `INSECURE_SKIP_VERIFY` | `false` | Skip TLS certificate verification (for self-signed certs) |

## Usage

```bash
# Generate dictionaries first (from the cisco-mobilitycore-pm-40 directory)
./generate-configurations-dictionaries.sh

# Upload to cloud
# Edit update-ingestion-dictionaries.sh to set BASE_URL, AUTHORIZATION_HEADER, etc.
./update-ingestion-dictionaries.sh
```

Both scripts use Docker (`golang:1.26.0-alpine3.23`) to run the Go tools — no local Go installation is required.

## What It Does

1. **Loads** all JSON dictionary files from `DICTIONARIES_PATH`
2. **Fetches** each corresponding dictionary from the server
3. **Compares** local vs remote (ignoring CouchDB metadata like `_rev`)
4. **Creates or Updates** dictionaries that differ

## API Endpoints Used

- `GET /api/v3/ingestion-dictionaries/{id}` - Fetch existing dictionary
- `POST /api/v3/ingestion-dictionaries` - Create new dictionary
- `PUT /api/v3/ingestion-dictionaries/{id}` - Update existing dictionary

## Prerequisites

- Docker (used to run Go tools via the `golang:1.26.0-alpine3.23` image — no local Go installation required)
